## 一、项目概述

### 1.1 项目目标

在一期基础上扩展系统功能：
1. **多作品支持** - 同时管理多部小说，不同题材/文风
2. **模板系统** - 预置题材模板，快速启动新项目
3. **人设管理** - 独立的人物设定管理模块
4. **世界观管理** - 功法、势力、地图等元素管理

### 1.2 技术栈

沿用一期技术栈：FastAPI + Vue3 + SQLite + Python 3.11+

## 二、需求汇总

### 2.1 功能需求

| 模块 | 功能 | 优先级 |
|-----|------|--------|
| 多作品支持 | 项目列表、切换、配置、归档 | P0 |

| 需求 | 目标 |
|-----|------|
| 兼容性 | 兼容一期数据，平滑升级 |
