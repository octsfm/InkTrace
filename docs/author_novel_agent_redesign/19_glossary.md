# 术语表

## 用途

统一定义系统核心术语，避免一个词在不同文档和代码里出现不同含义。

## 必须正式定义的词

- AuthorAgent
- StoryModel
- PlotArc
- ChapterMemory
- ContinuationMemory
- MemoryView
- ChapterPlan
- StructuralDraft
- FinalDraft
- RewriteDraft
- ValidationIssue
- Task
- Workflow
- AgentRun
- EffectiveVersion

## 规则

只要一个术语会影响代码结构、数据模型、状态语义或页面语义，就必须在这里先定义一次。
