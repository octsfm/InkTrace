## 一、项目概述

### 1.1 三期目标

引入**向量数据库 + RAG架构**，实现智能续写：
1. **智能检索** - 语义搜索相关章节、人物、伏笔
2. **上下文增强** - 自动组装最相关的上下文给LLM
3. **伏笔追踪** - 自动识别和提醒未回收的伏笔
4. **一致性保障** - 检索相关设定避免矛盾

---

## 二、技术方案汇总

### 2.1 核心技术

| 组件 | 技术选型 | 说明 |
|------|---------|------|
| 向量数据库 | ChromaDB | 本地轻量级，Python原生 |
| 嵌入模型 | text2vec-chinese | 中文优化，本地运行 |
| LLM | DeepSeek-V3 | 主力生成模型 |

### 2.2 RAG架构

```
用户请求 → 向量检索 → 上下文组装 → LLM生成 → 新章节
```
