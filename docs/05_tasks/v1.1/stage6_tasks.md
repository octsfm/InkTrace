### T6-04：今日新增字数实现

目标：

* 实现本地自然日维度的今日新增字数近似统计。

涉及模块：

* Frontend / Store

实现内容：

* 基于正文有效字符数变化计算今日新增。
* 按本地自然日维护 `todayKey`。
* 低于 0 时显示 0。
* 刷新后从本地恢复今日新增值。
* 不上传服务端。

关键规则：

* 今日新增字数为近似统计。
* 不作为强一致数据来源。
* 标题不计入今日新增字数。

完成标准（DoD）：

* 输入正文后今日新增增加。
* 删除文字后不显示负数。
* 刷新后今日新增值保持。

### T6-05：立即同步按钮实现

目标：

* 实现正文立即同步入口，触发当前正文草稿 flush。

涉及模块：

* Frontend / Store

实现内容：

* 实现 `ManualSyncButton`。
* 点击后触发当前章节标题与正文 flush。
* 同步时展示 saving 状态。
* 成功后展示 synced 状态。
* 离线或失败时保留本地草稿。

关键规则：

* 立即同步不改变自动保存机制。

* 失败时不得清理本地草稿。

完成标准（DoD）：

* 点击立即同步会发起保存请求。
* 成功后本地草稿清理。
* 失败或离线时草稿保留。
