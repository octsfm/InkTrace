## 四、API接口

### 4.1 向量索引接口

| 方法 | 路径 | 功能 |
|-----|------|------|
| POST | /api/novels/{novel_id}/vector/index | 索引小说内容 |
| GET | /api/novels/{novel_id}/vector/status | 获取索引状态 |
| DELETE | /api/novels/{novel_id}/vector/index | 删除索引 |

### 4.2 RAG检索接口

| 方法 | 路径 | 功能 |
|-----|------|------|
| POST | /api/novels/{novel_id}/rag/search | 语义搜索 |
| POST | /api/novels/{novel_id}/rag/context | 获取续写上下文 |
| POST | /api/novels/{novel_id}/rag/prompt | 生成RAG Prompt |
