# InkTrace V1.1 需求规格说明书（非 AI 写作增强，整理版）

版本：v1.1  
更新时间：2026-04-29  
范围定位：在 V1 纯文本写作主链路可用的前提下，补齐“纯写小说”刚需能力，并加入结构化写作辅助模块（不包含任何 AI 能力）。

---

## 1. 目标与原则

### 1.1 目标（结果导向）

- 正文区更干净：长时间写作不被 UI 噪音打断
- 不丢稿：Local‑First 自动保存可靠、可恢复
- 可组织：大纲/细纲、时间线、伏笔、人物卡支撑长篇写作
- 可管理：作品信息、章节定位、导入导出闭环

### 1.2 产品原则

- 正文区优先：中间正文区域只呈现“标题输入框 + 正文 + 必要状态”
- Local‑First：输入立即写本地缓存；debounce(2–3s) 同步；成功清缓存
- 单容器承载：写作辅助模块统一使用一个右侧抽屉容器（Tab 切换），避免多面板堆叠
- 非 AI：仅做 CRUD/排序/关联/跳转，不做任何生成/分析/推荐

---

## 2. 范围与分级

### 2.1 P0（必须交付：写作底座）

**P0-WS 写作页体验收口**
- 中间区极简：移除说明/调试占位；标题区不挤压正文可视高度
- 章节标题输入框（B2）：显示“第X章 + 标题”，可编辑；与正文共享 debounce 保存（同一次 flush 提交 title+content）
- 新建章节：无论当前选中哪章，均追加到最后一章；创建后自动选中、滚动聚焦、正文聚焦
- 章节编号：左侧列表始终显示“第X章”（X = order_index）
- Header：显示作品名（可选作者），不显示 UUID
- 右侧栏：V1.1 默认隐藏；仅在写作辅助抽屉启用时出现且只允许单一抽屉

**P0-WORK 作品信息管理闭环**
- 书架卡片 `(...)`：重命名作品、修改作者、删除（二次确认）
- 写作页可选入口：点击作品名快速编辑（可延后到 P1）

**P0-NAV 快速定位**
- 章节搜索：按标题关键词过滤（不改变真实顺序，仅影响展示）
- 跳转第 N 章：输入序号跳转并选中；无则提示
- 激活项可见：切章/新建/跳转后，列表 scrollIntoView 激活项

**P0-IO Web 导入导出闭环**
- TXT 导入（Web）：文件选择器 + 上传导入（multipart/form-data），禁止依赖 file_path
- TXT 导出：按 order_index 拼接；支持导出选项（含章节标题、章节间空行 0/1/2）
- 手动立即同步（可选）：提供“立即保存/立即同步”按钮触发 flush（不影响自动保存）

**P0-CH 章节小补丁**
- 新建章节模板：默认标题生成“第X章”（避免空标题）
- 长章节软提示：20 万有效字符软提示（已存在，保留）

### 2.2 P1（建议交付：写作组织模块，非 AI）

**P1-OUT 大纲（Outline）**
- 全书大纲（Work Outline）：文本模式必做；树状视图为增强（轻树，≤3 层）
- 章节细纲（Chapter Outline）：与章节绑定；切章自动切换
- 章节↔大纲关联（可选）：大纲节点手动关联章节用于跳转

**P1-TL 时间线（Timeline，顺序型）**
- 事件列表：标题、描述、顺序号
- 支持调整顺序（拖拽或上移/下移）
- 可选关联章节：点击跳转

**P1-FS 伏笔/线索（Foreshadow）**
- 记录：标题、描述、状态（未回收/已回收）
- 可选关联：引入章节/回收章节；默认视图优先未回收

**P1-CHAR 人物卡（最简）**
- 字段：name（必填）+ description（可选）
- 可选字段：aliases（别名/称呼）
- 支持搜索（name/aliases）

**P1-UX 编辑体验增强**
- 全屏/专注模式：隐藏侧边栏/顶部，仅保留状态条与正文
- 偏好：字体/字号/行距/主题（浅/深），本地持久化
- 今日新增字数：按日期累计有效字符新增（口径需确认）

## 3. 关键验收（DoD）

满足以下清单才算完成 V1.1：

- [ ] 写作页正文区极简：标题输入框 + 正文 + 底部轻量信息（无调试/说明占位）
- [ ] 新建章节永远追加到末尾；创建后选中并滚动聚焦；正文立即可写
- [ ] 左侧章节列表显示“第X章”（X=order_index）
- [ ] Header 显示作品名（可选作者），不显示 UUID
- [ ] 书架作品卡 `(...)` 支持重命名、改作者、删除（二次确认）
- [ ] 章节搜索与跳转第 N 章可用，且激活项始终可见
- [ ] Web TXT 导入可用（文件选择 + 上传导入）；导出按 order_index，且支持格式选项
- [ ] Local‑First 保存：输入立即写本地；debounce(2–3s) 同步；成功清缓存；刷新不丢稿

若包含 P1：
- [ ] 全书大纲与本章细纲（文本模式）可编辑、可保存、可随章节切换
- [ ] Timeline 事件列表可排序并可选关联章节跳转
- [ ] Foreshadow 支持未回收/已回收切换与章节关联跳转
- [ ] 人物卡可增删改查与搜索

## 4. 待确认（会影响实现，但不阻塞拆分）

1. Outline 树状视图最大层级（建议 ≤3）
2. Chapter Outline 是否与 Chapter 一对一（默认是一对一）
3. Timeline 事件是否允许关联多个章节（建议 0/1，避免复杂）
4. 今日新增字数统计口径（是否包含标题、是否按 flush 增量）
